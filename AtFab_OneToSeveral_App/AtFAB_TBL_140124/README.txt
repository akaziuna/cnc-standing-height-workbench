
ONE TO SEVERAL TABLE APP
=============================
'One to Several' parametric Processing table configuration app created by AtFab's (http://atfab.co). Distributed by Anna Kaziunas France with AtFab's permission. 

INSTALLATION
-----------------------------
For detailed instructions on how to install and use this app, see the CNC Maker Bench tutorial (http://makezine.com/cnc-maker-bench). 

DEPENDENCIES
-----------------------------
- Processing 2.1.1 (http://processing.org/)
- ControlP5 library 1.5.2 (http://www.sojamo.de/libraries/controlP5/)

DIRECTIONS AND TIPS
-----------------------------
